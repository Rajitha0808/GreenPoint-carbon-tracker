# GREENPOINT - Gamified Carbon Footprint Tracker

A sustainability platform that encourages urban youth to perform eco-friendly activities by rewarding them with green points, badges, and leaderboard rankings.

## Tech Stack

- **Frontend**: React (Vite), TailwindCSS, Lucide React, Recharts
- **Backend**: Node.js, Express
- **Database**: JSON (for prototype)

## Getting Started

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend
```bash
cd backend
npm install
npm start
```

## Features

- 🌱 Activity logging with points system
- 🏆 Gamification with badges and rewards
- 📊 Personal impact tracking with CO2 calculations
- 🏅 Leaderboard (weekly and all-time)
- 📱 Social sharing features
- 🌙 Dark mode support
- 📱 Responsive design
